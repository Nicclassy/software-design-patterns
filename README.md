# Software Design Patterns
This repository contains an implementation of various software design patterns in Kotlin.

The following design patterns will be implemented:

**Creational**
* Abstract Factory
* Factory Method
* Prototype
* Singleton

**Structural**
* Adapter
* Bridge
* Composite
* Decorator
* Facade
* Flyweight
* Proxy

**Behavioural**
* Chain of Responsibility
* Command
* Iterator
* Interpreter
* Memento
* Observer
* State
* Strategy
* Template Method
* Visitor
